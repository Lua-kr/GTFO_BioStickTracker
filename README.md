# Description
Add Bio Tags for Glowsticks to make them more useful gadgets to throw

## Config
### Only attach to glowsticks whose persistentID matches this value. Set 0 to work on all types of glowstick.
PersistentId = 0

### Scan radius around the glowstick (meters).
TagRadius = 3.0f

### Scan interval (seconds).
TickInterval = 0.2

### Tag Shadows?
TagShadows = false

### Tag Sleepers?
TagSleepers = true
